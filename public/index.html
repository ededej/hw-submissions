About Pranzo

Pranzo. Pranzo is a search application that searches lunch menus and will give the user a list of restaurants where they can eat lunch. The user will not necessarily need to register to use Lunchy but if they do the app will be able to remember user's choices and filter the results based on those choices and preferences. The user will need to insert a location (zip code) for the app to be able to display places that offer lunch
